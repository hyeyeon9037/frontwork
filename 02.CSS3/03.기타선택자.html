1. 속성 선택자

선택하고자 하는 요소 내에 작성 되어 있는 속성을 이용하여 선택하는 방법
선택자 뒤에 []을 이용하여 속성과 속성값을 제시하면서 선택 

선택자[속성=속성값] { 스타일속성 : 값; } → 속성=속성값 일때 선택
선택자[속성~=속성값] { 스타일속성 : 값; } → 속성값이 단어로 포함되어 있을 때(띄어쓰기로 구분함)
선택자[속성|=속성값] { 스타일속성 : 값; } → 속성값이 특정 값을 단어로 포함하고 있을 때 "-"로 구분
선택자[속성^=속성값] { 스타일속성 : 값; } → 속성값으로 시작하는 태그 선택
선택자[속성$=속성값] { 스타일속성 : 값; } → 속성값으로 끝나는 태그 선택
선택자[속성*=속성값] { 스타일속성 : 값; } → 속성값이 포함되어 있는 태그 선택

2. 자손선택자 후손선택자

요소들이 중첩되어 작성가능 자손 : 바로 하위인 요소들 후손 : 하위요소들 전부 

    * 자손선택자 : >
    a>b {
        스타일속성 : 값;
    }

    * 후손선택자 : 공백
    a b {
        스타일속성 : 값;
    }

3. 동위(같은레벨) 선택자

동위관계(같은레벨)에 있는 뒤에 위치한 특정 요소들 선택할 때 

    * a요소 뒤에 있는 b요소 "하나만" 선택
    a+b {
        스타일속성 : 값;
    }

    * a요소 뒤에 있는 모든 b요소 선택
    a~b {
        스타일속성 : 값;
    }

4. 반응 선택자

사용자의 움직임에 따라 선택되는 선택자

    *해당 요소에 클릭이 되었을 경우 스타일 부여
    선택자:active{
        스타일속성: 값;
    }

    *해당 요소에 마우스가 올라가는 순간 스타일 부여
    선택자:hover{
        스타일속성: 값;
    }

5. 상태 선택자

요소의 상태에 따라 선택되는 선택자 

        1) 체크된(checked) 상태의 요소내에
        선택자 : checked {
            스타일속성:값;
        }
    
        2) 초점(focus)가 맞춰진 input 요소 선택
        선택자 : focus {
            스타일속성:값;
        }

        선택자 : disabled {
