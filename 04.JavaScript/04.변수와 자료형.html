변수와 자료형

변수 선언 시 var뿐만 아니라 let, const를 이용하여 선언 가능

* var, let, const의 차이점

1) 변수 선언 방법

2) 스코프 (사용가능 범위)

* 변수 선언 위치에 따른 전역변수/ 지역변수

* 자료형

Dynamic Type : 변수 선언 시 자료형을 별도로 지정하지 않음 ( 자료형의 개념이 없는건 아니다. )
→ 변수 별로 타입이 지정되지 않고, 대입되는 값에 의해 타입이 결정된다.

자바스크립트의 자료형

  • string(문자열타입)
  • number(숫자)
  • boolean(논리값)
  • object(객체)
  • function(함수)
  • undefine(초기화가 안 된 변수)
