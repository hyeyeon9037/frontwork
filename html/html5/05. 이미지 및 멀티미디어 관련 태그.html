이미지 관련 태그

* src 속성


* alt 속성

    - 사진의 경로가 잘못되었거나 이미지를 제대로 표현하지 못할 경우 대체 텍스트의 용도
    - 시각 장애인들을 위한 스크린 리더(화면낭독기에서) 이미지를 읽어주는 설명 문구

* width와 height의 속성

    이미지의 가로, 세로 길이 조절을 하는 속성
    고정길이(px), 가변길이(%)로 지정 → 기본값은 px

고정 길이 단위(px) == 기본값 : 화면사이즈가 고정되어도 이미지의 크기는 변하지않는다.

가변길이단위(%) : 화면사이즈 또는 부모요소사이즈에 따라 이미지의 크기도 같이 변동
